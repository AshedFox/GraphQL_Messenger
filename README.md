# GraphQL Messenger

Real-time web-messenger based on Node.js, React and GraphQL
(with Apollo platform).

![App Screenshot](preview.png)

## How to run locally?

Clone the project

```bash
  git clone https://github.com/AshedFox/GraphQL_Messenger
```

Move to project directory

```bash
    cd ./GraphQL_Messenger
```

### To run server

Move to server subdirectory

```bash
    cd ./server
```

Install dependencies

```bash
    npm install
```

Build server

```bash
    npm run build
```

Start server

```bash
    npm run start
```

### To run client

Move to client subdirectory

```bash
    cd ./client
```

Install dependencies

```bash
    npm install
```

Start client

```bash
    npm run start
```
